1. Abstract

This document presents the user manual of the software developed as part of the Optimal Design Tools for Ocean Energy Arrays (DTOcean) EU research project. Its particular focus is on both a high level description of the different components that constituted the whole software and the Graphical User Interface and its related functionalities.
